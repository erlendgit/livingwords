Waar gaat dit over?
===

Het is een bijbel studie tool. Het doel is om zelf gevonden relaties te leggen tussen bijbel teksten.

Bij het overnemen van de Bijbelteksten doe je het volgende:

## 1. Verzamel ingredienten

### Woord

Dit is wat je leest in de Bijbel. Het doel van de interface is om de woorden te kunnen overnemen en van context te
kunnen voorzien. Dan kan je later via de context terug bij de tekst komen.

Een woord heeft kenmerken:

* Inhoud
* Boek
* Hoofdstuk
* Vers
* Notities (vragen die ik zelf over de tekst wil stellen)

### Ingredienten

* Verhaal; In welk verhaal zitten we
* Acteurs; schrijver, verteller, luisteraar
* Context; waar het verhaal zich afspeelt
* Vraag; Wat voor vraag wordt beantwoord in de tekst
* Waarheid; Welke waarheidsclaim wordt hier gemaakt

## Interface

* Centraal staat een invoerveld voor de inhoud van het vers.
  * Boven het vers staan de 5 eerdere versen en een context vergroter.
* Onder het vers staan invoervelden voor de kenmerken en ingrediënten.
  * Een ingrediënt wordt getoond als
    * Het actieve ingrediënt
    * Een zoekfilter
    * Een lijst met bestaande ingredienten van dat type, en een selecteer optie
  * Je kunt er kiezen uit:
    * Welk verhaal / welke verhalen zitten we in
    * Wie is de verteller
    * Wie is de spreker
    * Wie is de luisteraar (1 of meer)
    * Wie zijn omstanders, indien van toepassing.
    * Welke context zit het verhaal in
    * Welke geestelijke waarheid is van toepassing (1 of meer)
    * Welke vraag wordt beantwoord (1 of meer)
